# Patient Billing Tracker

The Patient Bill Tracker is an application designed to help healthcare organizations track, manage, and monitor patient billing information throughout the revenue cycle. It centralizes billing data, payment status, outstanding balances, and billing-related events into a single interface, making it easier to manage patient accounts and financial workflows.

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Lucide-react
- Vite

## Features

- Dynamic payment status derived from data
- Live search and filter working together
- Real-time summary dashboard
- Progress bars showing payment completion
- Responsive grid layout

## Run Locally

cd patient-billing-tracker
